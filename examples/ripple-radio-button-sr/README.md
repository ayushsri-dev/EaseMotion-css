# Ripple Radio Button

A simple and reusable radio button component with a smooth ripple animation. It is designed for blog pages, forms, and content filters where users need a clean and interactive way to choose one option.

---

## Features

- Pure HTML and CSS
- No JavaScript required
- Smooth ripple animation
- Responsive layout
- Accessible and keyboard-friendly
- Easy to customize
- Works with EaseMotion CSS utility classes

## How to Use

Add the following markup to your HTML file:

```html
<label class="radio-item ease-hover-lift">

    <input type="radio" name="category">

    <span class="custom-radio"></span>

    <span class="label-text">
        Technology
    </span>

</label>
```

Open `demo.html` in any modern browser to see the component in action.

---

## Why This Component?

Radio buttons are commonly used for selecting a single option. This version adds a small ripple animation to provide clear visual feedback while keeping the design clean and professional. It is suitable for blog category filters, forms, surveys, and settings pages.
